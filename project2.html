Shoe Collection Management Program

Manage a personal shoe inventory: add shoes, remove shoes, and sort by price to compare values at a glance.

Description

This project focused on building a program that manages a personal shoe collection. Users can add new shoes, remove shoes, and organize their collection by features such as price. The program can display shoes sorted from high to low, helping users keep track of what they own and compare values. It simulates the idea of a digital shoe inventory system.

Skills Learned

  • Designing and managing data structures for collections
  • Implementing add/remove functionality with user input
  • Sorting data by price and other criteria
  • Practicing object-oriented programming ideas (classes and objects)
  • Strengthening problem-solving and logical thinking through coding
